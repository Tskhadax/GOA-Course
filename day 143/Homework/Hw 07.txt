7) ბონუსად შეასრულეთ ეს დავალება და ატვირთეთ თეორია(თეორია რაც არის დასაწერი ისინი ერთ txt ფაილში ჩამოწერეთ) თქვენს GOA-HOMEWORK რეპოზიტორიაში

Git-ის ხშირად გამოყენებადი ბრძანებები:

git init - ბრძანება, რითაც ქმნი გითის ახალ ცარიელ რეპოზიტორიას. ასევე ქმნის გითის ფაილს (.git) და იქ ინახება ფაილების შესახებ ინფორმაცია, მათ შორის ცვლილებების ისტორიაც.
git clone {აქ იქნება რეპოზიტორიის მისამართი} - აკლონებს/აკოპირებს/იწერს უკვე არსებული რეპოზიტორიის ინფორმაციას. git clone-ის მერე ვწერთ რეპოზიტორიის მისამართს რისი გადმოწერაც გვინდა.
git add - მისი საშუალებით ვამატებთ ფაილებს ჩვენს რეპოზიტორიაში. 
    დამხმარე პარამეტრები:
        git add . - ყველა ფაილს დაამატებს სადაც ნახავს ცვლილებას. წაშლას, სახელის შეცვლას, ახალ ფაილს და ა.შ. მისი სინონიმია git add -A.
        git add --ignore-removal - დაამატებს ახალ ან შეცვლილ ფაილებს. წაშლილ ფაილებს არ შეეხება.
        git add -u - შეცვლილ და წაშლილ ფაილებს შეეხება. ახალს არა.
            ან შეგვიძლია უბრალოდ ფაილის სახელი დავწეროთ გვერდით
        git add {ფაილის მისამართი რისი დამატებაც გვინდა მარტო}.
git status - აჩვენებს ახლა რომელ ფაილშიც ვართ იმის სტატუს. ანუ სადმე რამე ცვლილება თუ არის და არ არის ატვირთული და ა.შ...
git diff - აჩვენებს რა სახის ცვლილება და რა დაემატა იქ სადაც ვიმყოფებით.
git commit -m" Title" -ასატვირთად გამზადებული ფაილების სათაურს ვწერთ. ამ ატვირთვის/მიმდინარე მდგომარეობის სათაურს.
git checkout - გადადის ახალ branch-ზე. სინონიმია git switch
    git checkout -b 'branch name' - ქმნის და გადადის ახალ branch-ზე. ===  git switch - c 'branch name'
